# Asha Baker Instructional Design Portfolio

This folder contains a complete static HTML portfolio website and downloadable portfolio artifacts.

## Files included
- `index.html` - home page
- `projects.html` - project cards
- `sample-*.html` - interactive prototypes
- `artifacts/` - downloadable PDF instructional design samples
- `resume/` - resume PDF
- `Asha_Baker_Instructional_Design_Portfolio.pdf` - printable portfolio PDF
- `assets/` - CSS and JavaScript

## Publish with GitHub Pages
1. Create a GitHub account or log in.
2. Create a new public repository named something like `asha-baker-id-portfolio`.
3. Upload every file and folder from this portfolio folder into the repository root.
4. Go to Settings > Pages.
5. Under Build and deployment, choose Deploy from a branch.
6. Select the `main` branch and `/root` folder.
7. Save and wait for GitHub to publish the site.
8. Your portfolio link will look like: `https://YOUR-USERNAME.github.io/asha-baker-id-portfolio/`.
